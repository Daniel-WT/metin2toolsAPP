Configurare Firebase

Firebase iti permite sa sincronizezi datele cu alti utilizatori in timp real. Urmeaza pasii de mai jos — dureaza ~5 minute.

1
Mergi la console.firebase.google.com si autentifica-te cu contul Google.
2
Apasa „Add project" → da un nume (ex: metin2-tools) → dezactiveaza Google Analytics → Create project.
3
In proiect, apasa pictograma </> (Web) → da un nume aplicatiei → apasa Register app. Vei vedea un obiect firebaseConfig — copiaza valorile de acolo jos.
4
Din meniu stanga → Build → Realtime Database → Create database → alege locatia (europe-west1 recomandat) → Start in test mode → Enable.
5
Copiaza URL-ul bazei de date (arata ca https://metin2-tools-default-rtdb.europe-west1.firebasedatabase.app) si introdu-l mai jos.
